# Picture-eraser

Picture-Eraser allows users to resize、crop、erase picture.
It makes user easily to removing and crop any part of a picturer。

## preview

https://lawliet01.github.io/picture-eraser/

## feature:
1. Picture Input
2. Picture resize
3. Picture crop
4. Picture erase
5. Picture reset
6. Change undo
7. Picture download
